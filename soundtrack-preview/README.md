# Soundtrack Audition

Serve the repository root and open `soundtrack-preview/index.html`. The page
uses the generated `web/music-data.js` event tables directly, so it cannot drift
from the bank shipped with the DOS and browser builds.

The five playback modes are compact Web Audio character models for comparison,
not recordings of proprietary hardware. Native hardware initialization and
sequencing are verified through `/audiodump` under DOSBox-X or real hardware.
